# SkyFlow
SkyFlow is a mobile-responsive weather application built using HTML, CSS, and JavaScript. It provides real-time weather information for any city with animated backgrounds that change according to weather conditions. The app features a clean UI, smooth animations, and work seamlessly across mobile and desktop devices.
